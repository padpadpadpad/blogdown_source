Over the last 4 years I have consistently cycled around or over 2000 km a year around Cornwall. Almost all of these rides have been uploaded to Strava which has a suite of nice ways to visualise your performance and each ride.

However, the ride data on Strava is mine, I have literally created it, and as such I looked for a way of accessing all of that beautiful data. One way would be to download all your .gpx files and read them in. That is going to be a lot of data. The alternative I settled on is rStrava, an R package which wraps the Strava API in many easy-to-use functions allowing access to all your glorious data.

desired_cols <- desired_columns <- c('achievement_count', 'athlete.id', 'average_cadence', 'average_heartrate', 'average_speed', 'average_watts', 'distance', 'elapsed_time', 'elev_high', 'elev_low', 'kilojoules', 'suffer_score', 'max_heartrate', 'max_speed', 'moving_time', 'start_date', 'start_date_local', 'name', 'total_elevation_gain', 'type', 'map.summary_polyline', 'location_city', 'upload_id')

# delete data with a distance < 3 km
my_acts <- filter(my_acts, distance > 3 & type != 'Workout')

# transformations of data
my_acts <- mutate_at(my_acts, c('average_cadence', 'average_heartrate', 'average_speed', 'average_watts', 'distance', 'elapsed_time', 'elev_high', 'elev_low', 'kilojoules', 'suffer_score', 'max_heartrate', 'max_speed', 'moving_time', 'total_elevation_gain'), as.numeric) %>%
  mutate(.,
         activity_no = seq(1,nrow(.), 1),
         elapsed_time = elapsed_time/60/60,
         moving_time = moving_time/60/60,
         elevation_per_km = total_elevation_gain/distance,
         date = gsub("T.*$", '', start_date) %>%
           as.POSIXct(., format = '%Y-%m-%d'),
         EUdate = format(date, '%d/%m/%Y'),
         month = format(date, "%m"),
         day = format(date, "%d"),
         year = format(date, "%Y")) %>%
  mutate_at(., c('month', 'day', 'year'), as.numeric) %>%
  arrange(., date) %>%
  group_by(year, type) %>%
  mutate(cum_distance = cumsum(distance)) %>%
  data.frame()
